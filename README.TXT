INSTALLATION
-------------------
 
* Installation is simple change base_url  and database credentials in 
   -> application/config/config.php
   -> application/config/database.php
*  check constant values in 
    -> application/config/constants.php

--php.ini--

*** Please set max_execution_time    // depend on data 
*** Please set max_upload_file_size  // 20mb
*** Please set max_post_size            // 100mb

DEPENDENCIES
--------------------

1) Firebase (for chatting )
2) Twillo (for sending sms )
4) Facebook api to login 
5) Gmail api to login 
6) Google map credentials
7) Mango Pay payment gateway
